Embed DoBu.uk availability and booking calendars

== Description ==

Provides a block which embeds a [DoBu.uk availability calendar and enquiry form](https://dobu.uk)
for a given nick name.

See the above link for a demonstration.

== Screenshots ==

1. Mobile view of the embedded calendar.
2. Tablet view of the embedded calendar.
3. Desktop view of the calender (requires a widescreen theme).
4. Adding the calendar.

== Installation ==

This plugin is can be installed with one-click through the block
directory. Below are manual instructions.

1. Unzip `dobuuk-availability.zip` into the `/wp-content/plugins/dobuuk-availability` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.

Once installed it should appear in the block editor by searching for
"DoBu.uk availability", or less.
